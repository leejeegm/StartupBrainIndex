# 한글 PDF 폰트 (선택)

PDF 보고서에서 한글이 깨지지 않도록 아래 중 하나를 이 폴더에 넣어 주세요.

- **NanumGothic.ttf** — [나눔고딕](https://github.com/naver/nanumsquare/raw/master/NanumGothic.ttf) 다운로드 후 이 폴더에 저장
- **NotoSansKR-Regular.ttf** — [Google Noto Sans KR](https://fonts.google.com/noto/specimen/Noto+Sans+KR) 에서 받아 저장

Windows에서는 맑은고딕(malgun.ttf)을 자동으로 사용합니다.  
Linux(Render 등)에서는 위 폰트 파일을 추가하면 한글이 정상 출력됩니다.
